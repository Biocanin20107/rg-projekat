# Kontrole
1. WASD - standardno pomeranje kamere
2. I - podizanje kamere
3. J - spustanje kamere
4. B - blinn on/off
5. SPACE - HDR on/off
6. Q - exposure -0.05
7. E - exposure +0.05
8. UPARROW - okretanje kamere na gore
9. DOWNARROW - okretanje kamere na dole
10. P - camera mouse update on/off
11. F1 - IMGUI on/off
